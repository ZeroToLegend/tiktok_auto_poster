# Verified Hooks — Coding Niche

Hooks đã được test và có engagement tốt. Dùng làm reference cho skill `tiktok-hook-writer`.

Format: `Framework | Hook | Context`

## Number

- `Number | "7 shortcut VS Code tiết kiệm 3 tiếng/ngày" | productivity`
- `Number | "3 dòng Python auto 90% task Excel" | automation`
- `Number | "13 lỗi Python mà intern nào cũng mắc" | mistakes`
- `Number | "5 extension VS Code phải cài ngày đầu đi làm" | tools`

## POV

- `POV | "POV: sếp hỏi bạn biết SQL và bạn vừa Google 5 phút trước" | career relatable`
- `POV | "POV: lần đầu deploy Friday chiều" | humor/pain`
- `POV | "POV: code review junior viết if/else 7 tầng" | code quality`

## Contrarian

- `Contrarian | "Đừng học JavaScript đầu tiên" | learning path`
- `Contrarian | "Python không phải ngôn ngữ dễ nhất để bắt đầu" | opinion`
- `Contrarian | "Bootcamp 3 tháng không biến bạn thành senior" | career reality`

## Pain/Problem

- `Pain | "Code đúng nhưng vẫn bị reject phỏng vấn? Đây là lý do" | interview`
- `Pain | "Học 6 tháng vẫn chưa làm được project cá nhân?" | stuck learner`
- `Pain | "Pandas chậm? Check 4 điểm này trước khi đổi tool" | performance`

## Question

- `Question | "Tại sao 99% dev viết if x == True sai?" | code quality`
- `Question | "Sao dùng Docker production lại sập nặng?" | ops`
- `Question | "Git rebase hay merge? Câu trả lời cuối cùng" | opinion`

## Curiosity

- `Curiosity | "Mẹo này làm code nhanh gấp 10 mà trường không dạy" | productivity`
- `Curiosity | "Tôi phát hiện lý do thật sự code lag trên M1" | tech deep dive`

## Outcome

- `Outcome | "Từ 0 thành backend dev 14 tháng, đây là roadmap" | career`
- `Outcome | "Tôi tự build script 50 dòng, auto cả team data" | automation`

---

## Notes

- Hook với số lẻ (3, 7, 13) convert tốt hơn số chẵn
- POV hook viral nhất khi audience cảm thấy "mình đây"
- Contrarian chỉ work nếu có argument backup trong video
- Tránh hook "learn Python in X days" — đã bão hòa
